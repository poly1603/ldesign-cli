## 📝 交付说明

### 安装要求
```
Node.js:    >= 18.0.0
pnpm:       >= 8.0.0
系统:       Windows/macOS/Linux
```

### 首次安装
```bash
# 1. 安装依赖
pnpm install

# 2. 安装 sql.js (重要!)
pnpm add sql.js

# 3. 构建项目
pnpm build

# 4. 启动测试
ldesign ui
```

### 注意事项
1. 需要安装 `sql.js` 依赖
2. 首次启动会自动初始化数据库
3. 会在 `~/.ldesign-cli` 创建配置目录
4. 插件目录在 `~/.ldesign-cli/plugins`
